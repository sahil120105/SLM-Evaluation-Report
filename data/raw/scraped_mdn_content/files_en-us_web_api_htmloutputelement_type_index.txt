# HTMLOutputElement: type property

{{ApiRef("HTML DOM")}}

The **`type`** read-only property of the {{domxref("HTMLOutputElement")}} interface returns the string `"output"`.

## Value

The string `"output"`.

## Example

```js
const output = document.querySelector("output");
console.log(output.type); // "output"
```
