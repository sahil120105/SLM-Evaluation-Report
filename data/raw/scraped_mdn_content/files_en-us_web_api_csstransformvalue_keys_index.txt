# CSSTransformValue: keys() method

{{APIRef("CSS Typed OM")}}

The **`CSSTransformValue.keys()`** method
returns a new _array iterator_ object that contains the keys
for each index in the array.

## Syntax

```js-nolint
keys()
```

### Parameters

None.

### Return value

A new {{jsxref("Array")}}.
