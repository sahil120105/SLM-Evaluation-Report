# CustomStateSet: clear() method

{{APIRef("Web Components")}}

The **`clear()`** method of the {{domxref("CustomStateSet")}} interface removes all elements from the `CustomStateSet` object.

## Syntax

```js-nolint
clear()
```

### Parameters

None.

### Return value

Undefined.
