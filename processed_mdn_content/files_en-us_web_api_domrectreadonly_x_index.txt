# DOMRectReadOnly: x property

{{APIRef("Geometry Interfaces")}}{{AvailableInWorkers}}

The **`x`** read-only property of the **`DOMRectReadOnly`** interface represents the x coordinate of the `DOMRect`'s origin.

## Value

A double.
