# SVGComponentTransferFunctionElement: slope property

{{APIRef("SVG")}}

The **`slope`** read-only property of the {{domxref("SVGComponentTransferFunctionElement")}} interface reflects the {{SVGAttr("slope")}} attribute of the given element.
