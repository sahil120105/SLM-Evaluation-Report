# CSSTransformValue: values() method

{{APIRef("CSS Typed OM")}}

The **`CSSTransformValue.values()`** returns a
new _array iterator_ object that contains the values for
each index in the CSSTransformValue object.

## Syntax

```js-nolint
values()
```

### Parameters

None.

### Return value

A new {{jsxref("Array")}}.
