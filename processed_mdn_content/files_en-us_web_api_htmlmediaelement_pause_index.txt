# HTMLMediaElement: pause() method

{{APIRef("HTML DOM")}}

The **`HTMLMediaElement.pause()`** method will pause playback
of the media, if the media is already in a paused state this method will have no effect.

## Syntax

```js-nolint
pause()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).

### Exceptions

None.
