# MIDIOutput: clear() method

{{securecontext_header}}{{APIRef("Web MIDI API")}}

The **`clear()`** method of the {{domxref("MIDIOutput")}} interface clears the queue of messages being sent to the output device.

## Syntax

```js-nolint
clear()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
